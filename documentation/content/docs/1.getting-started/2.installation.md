# Installation

TestFlowKit is a standalone executable that requires no complex installation process. Download the appropriate version for your operating system and you're ready to go.

## Prerequisites

Before installing TestFlowKit, ensure you have:

- ℹ️ A modern web browser (Chrome or Edge) for frontend testing
- ℹ️ A computer running Windows, macOS, or Linux

## Download

Download the latest version of TestFlowKit for your platform from the [GitHub Releases](https://github.com/TestFlowKit/testflowkit/releases) page.

### Available Platforms

| Platform | Architecture | File |
|----------|--------------|------|
| Windows | x64 | `tkit-windows-amd64.zip` |
| Windows | ARM64 | `tkit-windows-arm64.zip` |
| macOS | Intel | `tkit-darwin-amd64.tar.gz` |
| macOS | Apple Silicon | `tkit-darwin-arm64.tar.gz` |
| Linux | x64 | `tkit-linux-amd64.tar.gz` |
| Linux | ARM64 | `tkit-linux-arm64.tar.gz` |

## Setup Instructions

### Windows

1. Download the appropriate ZIP file for your architecture
2. Extract the ZIP file to a folder of your choice
3. Open Command Prompt or PowerShell
4. Navigate to the folder containing `tkit.exe`
5. Run `.\tkit.exe --version` to verify the installation

### macOS

1. Download the appropriate tar.gz file for your Mac
2. Extract the archive:
   ```bash
   tar -xzf tkit-darwin-*.tar.gz
   ```
3. Make the binary executable:
   ```bash
   chmod +x tkit
   ```
4. Verify the installation:
   ```bash
   ./tkit --version
   ```

> ⚠️ **macOS Security:** On first run, macOS may block the application. Go to **System Preferences > Security & Privacy** and click "Allow Anyway" to permit TestFlowKit to run.

### Linux

1. Download the appropriate tar.gz file for your distribution
2. Extract the archive:
   ```bash
   tar -xzf tkit-linux-*.tar.gz
   ```
3. Make the binary executable:
   ```bash
   chmod +x tkit
   ```
4. Optionally, move to a directory in your PATH:
   ```bash
   sudo mv tkit /usr/local/bin/
   ```
5. Verify the installation:
   ```bash
   tkit --version
   ```

## Verify Installation

After installation, verify that TestFlowKit is working correctly:

```bash
tkit --version
```

You should see output similar to:

```
TestFlowKit version 2.0.0
```

## Next Steps

Now that TestFlowKit is installed, continue to the [Quick Start](/docs/getting-started/quick-start) guide to write your first test.
